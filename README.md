# tauaminibar
este sistema visa reduzir perdas de produtos do frigobar e otimizar o processo de cobrança, permitindo que o hóspede adicione consumos após a primeira conferência no check-out. Benefícios do Sistema:

Redução de perdas de produtos do frigobar. Otimização do processo de cobrança. Maior transparência e controle sobre os consumos. Melhoria na experiência do hóspede. Aumento da receita do hotel.
